DoDo Framework

Start a new single page app project with DoDo Framework.

  $ git clone -b master https://github.com/miko-soft/create-dodo-boilerplates.git [projectName]
  $ cd [projectName]
  $ rm -rf .git && git init
  $ npm install
or just 
$ npm init create-dodo [projectName]


Now build your app with vanilla HTML, SCSS and JS files.
The documentation at http://dodo.mikosoft.info .
